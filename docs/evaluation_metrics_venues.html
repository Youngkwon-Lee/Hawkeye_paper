2. QWK (Quadratic Weighted Kappa) — Intuitive Explanation

Definition: "두 평가자(VLM vs Expert)가 얼마나 일치하는지" 측정. 가까이 틀린 건 봐주고, 멀리 틀린 건 크게 벌점을 주는 방식.

Penalty Matrix (차이² = Quadratic)

실제 점수와 예측 점수의 차이에 따른 벌점:

Pred 0Pred 1Pred 2Pred 3Pred 4
True 0014916
True 110149
True 241014
True 394101
True 4169410

Why QWK over Accuracy?

Accuracy (Exact Match)

True=2, Pred=3 → WRONG (0 points)
True=2, Pred=0 → WRONG (0 points)
→ 1점 차이와 2점 차이를 동일하게 취급!

QWK (Quadratic Weighted)

True=2, Pred=3 → Penalty 1 (경미)
True=2, Pred=0 → Penalty 4 (심각)
→ 임상적 심각도에 비례하는 벌점!

0.6-0.8
0.8-1.0
QWK RangeInterpretationReference
< 0.20거의 일치 안 함Random guess
0.21 - 0.40약한 일치 (Fair)
0.41 - 0.60보통 일치 (Moderate)VLM zero-shot 목표
0.61 - 0.80상당한 일치 (Substantial)MDS-UPDRS 의사 간 수준
0.81 - 1.00거의 완벽 (Excellent)Training set CORAL
임상 맥락: MDS-UPDRS 의사 간 QWK가 보통 0.6-0.8. VLM이 0.4+ 나오면 "의미 있다", 0.6+ 나오면 "임상 수준에 근접"이라는 스토리.

QWK Formula

QWK = 1 - (Σ wij × Oij) / (Σ wij × Eij)

wij = (i - j)² / (K - 1)²    (weight matrix, K=5 classes)
Oij = observed confusion matrix
Eij = expected (random chance) confusion matrix

→ O가 E보다 좋으면 QWK > 0 (찍기보다 나음)
→ 완벽 일치이면 QWK = 1

3. Domain-Specific Metric Standards

DomainPrimary MetricStandard InPapers Using It
AQA (Action Quality Assessment) Spearman ρ CoRe, TSA, MUSDL, MCoRe 56/96 papers (58%)
MDS-UPDRS Clinical ICC / QWK MDS official, clinical trials Gold standard for inter-rater
VLM Benchmarks (Video-MME etc) Accuracy (MCQ) Video-MME, MVBench, EgoSchema Multiple choice format
Ordinal Regression MA-MAE + Spearman Medical ordinal, class-imbalanced Recent best practice (2024+)
Our Current Models (CORAL) Pearson + MAE + OBO Hawkeye codebase Needs Spearman/QWK addition
Key Insight — MA-MAE vs MAE:
PD4T score distribution: Score 0 (~50), Score 1 (~60), Score 2 (~50), Score 3 (~30), Score 4 (~10)
Regular MAE: Score 4를 다 틀려도 MAE에 거의 영향 없음 (10/200 = 5%)
MA-MAE: 클래스별 MAE를 구한 뒤 평균 → Score 4 성능이 동등하게 반영됨. 불균형 데이터에서 필수.
